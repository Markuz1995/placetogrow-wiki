ChangeLog

Descripción: Esta versión introduce la implementación completa de operaciones CRUD para Micrositios y Categorías. Las características clave y las actualizaciones incluyen:

Nuevas Funcionalidades

  • CRUD de Micrositios:
    • Operaciones de Crear, Leer, Actualizar y Eliminar para gestionar micrositios.
    • Integración con los frameworks de front-end Inertia.js y React.js.
    • Soporte para carga y eliminación de imágenes.
  • CRUD de Categorías:
    • Operaciones de Crear, Leer, Actualizar y Eliminar para gestionar categorías.
    • Selección de categorías integrada en los formularios de creación y actualización de micrositios.

Mejoras

  • Refinamiento del código siguiendo los principios SOLID y estándares PSR.
  • Mejora de la experiencia del usuario con validaciones intuitivas de formularios y manejo de errores.
  • Corrección de varios errores menores relacionados con envíos de formularios y validaciones de datos.

Detalles Técnicos

  • Frameworks y Bibliotecas: Laravel 11, PHP 8.2, Inertia.js, React.js, Tailwind CSS.
  • Migraciones de Base de Datos: Incluye migraciones necesarias para nuevas tablas y relaciones.

Documentación de API ("/api")

Módulo Categoría

El módulo "Categoría" sirve como un sistema de clasificación para los micrositios en la plataforma. Su principal función es organizar y etiquetar los micrositios según sus características. Esto facilita la navegación, búsqueda y gestión de los micrositios tanto para los administradores como para los usuarios.

Funcionalidad Descripción Método Ruta Respuesta
Listar categorías Recupera una lista de todas las categorías. GET /category Inertia: 'Category/Index' con datos de categorías y mensaje de éxito
Crear categoría Crea una nueva categoría. POST /category Redirección a 'category.index' con mensaje de éxito
Obtener categoría Recupera detalles de una categoría específica. GET /category/{id} Inertia: 'Category/Show' con datos de la categoría
Actualizar categoría Actualiza una categoria existente. PUT /category/{id} Redirección a 'category.index' con mensaje de éxito
Eliminar categoría Elimina una categoria existente. DELETE /category/{id} Redirección a 'category.index' con mensaje de éxito

Módulo Micrositio

Los micrositios son pequeñas plataformas que los usuarios pueden registrar dentro de la aplicación. Cada micrositio permite a los usuarios gestionar su propia información y ofrece funcionalidades específicas relacionadas con diferentes tipos de pagos.

Funcionalidad Descripción Método Ruta Respuesta
Listar micrositios Recupera una lista de todas los micrositio. GET /microsites Inertia: 'Microsite/Index' con datos de micrositios y mensaje de éxito
Crear micrositio Crea un nuevo micrositio. POST /microsites Redirección a 'microsite.index' con mensaje de éxito
Obtener micrositio Recupera detalles de un micrositio específico. GET /microsites/{id} Inertia: 'Microsite/Show' con datos del micrositio
Actualizar micrositio Actualiza un micrositio existente. PUT /microsites/{id} Redirección a 'microsite.index' con mensaje de éxito
Eliminar micrositio Elimina un micrositio existente. DELETE /microsites/{id} Redirección a 'microsite.index' con mensaje de éxito
